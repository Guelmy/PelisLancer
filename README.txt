# PelisLancer
Esta es la aplicación  de la prueba técnica
para instalarlo debe tener instalado en el ordenador PYTHON, PostgreSQL, Django
para poner usar la base de datos postgres se debe instalar el controlador o driver Psycopg2 (solo si 
se instalara la aplicacón en el sistema operativo Windows.

En caso de no encontral el driver aqui le dejo un link donde puede descargarlo http://www.stickpeople.com/projects/python/win-psycopg/index.html

Se debe crear un proyecto y ejecutar el servidor.

Una vez en el servidor este corriendo se abrira la  Web Application en la intefaz de login de Django Se debera ingresar el usuario y la contraseña siguientes: Usuario: admin, Passoword: TheMaster0124.

Una vez verificada la información se abrira la página de admin de Django en la que se puede agregar Usuarios o Clientes,
Tambien se puede ingresar peliculas, se realizara una lista automaticamente con las peliculas existentes, se puede eliminar una pelicula que ya no este disponible y se puede modificar
